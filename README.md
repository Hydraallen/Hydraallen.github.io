## Usage

### config page

edit `_config.yml`

### New page

```bash
cd .../Hydraallen.github.io/my-blog/source/_posts

hexo new post "New page title"
```

### test

```bash
hexo g && hexo s
```

### Update

```bash
hexo deploy
```

### Clean
```bash
hexo clean
```
